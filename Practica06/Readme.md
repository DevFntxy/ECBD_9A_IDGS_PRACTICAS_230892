# Sistema de Predicción de Riesgo Coronario — Estado de Puebla
## Unidad III: Integración del Equipo y Matriz de Roles (Enfoque en Machine Learning Clínico)

Este documento detalla la estructura organizativa del **Equipo MR** para el desarrollo de una plataforma inteligente orientada a la evaluación y predicción del riesgo coronario (cardiovascular) en pacientes del Estado de Puebla. La distribución de roles técnicos garantiza un flujo óptimo desde la ingesta de expedientes médicos hasta el despliegue del modelo predictivo y la visualización epidemiológica.
